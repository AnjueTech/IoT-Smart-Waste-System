# IoT-Smart-Waste-System
**IoT-Smart-Waste-System** uses **ESP8266, Arduino Uno, DHT11, and Ultrasonic Sensors** to monitor bin levels and environment data. It features **real-time tracking via a web app (PHP, MySQL, JS)** with **Google Maps integration** for remote bin location, optimizing waste collection, cutting costs, and preventing pollution. 🚀
